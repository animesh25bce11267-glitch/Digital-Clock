# Aesthetic Digital Clock (Python + Tkinter)

A simple and beautifully designed Digital Clock Application built using Python and the Tkinter GUI library.

This project displays:

Current Time (HH:MM:SS AM/PM)

Current Day

Current Date

The interface features a modern dark theme with neon-accented text for an aesthetic, minimal look.

# Features

Real-time time updates

Clean and aesthetic UI design

Shows time, day, and full date

Beginner-friendly Python project

Lightweight and runs as a standalone desktop app

# How It Works

The program uses:

time.strftime() to fetch the live system time

Tkinter labels to display time, day, and date

A root.after() loop to refresh the clock every second

# How to Run

Install Python

Download the file clock.py

Open a terminal in the project folder

Run:

python clock.py
